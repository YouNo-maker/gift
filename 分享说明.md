# 生日网页分享说明

## 📦 方案一：打包成 ZIP 文件发送（推荐）

### 步骤：
1. **手动打包**：
   - 选中以下所有文件和文件夹：
     - `index.html`
     - `css/` 文件夹（包含所有CSS和JS文件）
     - `music/` 文件夹（包含音频文件）
     - `生日蛋糕.png`
   - 右键 → 发送到 → 压缩(zipped)文件夹
   - 或者使用 WinRAR/7-Zip 等工具打包

2. **发送给朋友**：
   - 将生成的 ZIP 文件通过微信、QQ、邮箱等方式发送

3. **朋友使用**：
   - 解压 ZIP 文件到任意文件夹
   - 双击 `index.html` 文件即可在浏览器中打开
   - **注意**：必须保持文件夹结构不变（css/ 和 music/ 文件夹要在 index.html 同级目录）

---

## 🌐 方案二：部署到 GitHub Pages（免费在线访问）

### 步骤：
1. 在 GitHub 创建新仓库（设为 Public）
2. 上传所有文件到仓库
3. 在仓库设置中启用 GitHub Pages
4. 将生成的链接分享给朋友

**优点**：朋友可以直接通过链接访问，无需下载

---

## 💻 方案三：使用本地服务器（适合技术用户）

如果直接打开 HTML 文件遇到跨域问题，可以使用本地服务器：

```bash
# 使用 Python（如果已安装）
python -m http.server 8000

# 或使用 Node.js 的 http-server
npx http-server
```

然后访问 `http://localhost:8000`

---

## ⚠️ 重要提示

1. **文件结构必须保持**：
   ```
   birthday/
   ├── index.html
   ├── css/
   │   ├── default.css
   │   ├── functions.js
   │   ├── love.js
   │   └── ... (其他JS文件)
   ├── music/
   │   ├── love.mp3
   │   └── 杨和苏KeyNG - 很久很久.flac
   └── 生日蛋糕.png
   ```

2. **浏览器兼容性**：
   - 推荐使用 Chrome、Edge、Firefox 等现代浏览器
   - 移动端建议使用微信内置浏览器或 Safari

3. **音频文件**：
   - 如果音频无法播放，检查浏览器是否支持 FLAC 格式
   - 建议保留 MP3 作为备选格式（代码中已包含）

---

## 🎁 快速打包命令（Windows PowerShell）

如果文件没有被占用，可以运行：

```powershell
Compress-Archive -Path index.html,css,music,"生日蛋糕.png" -DestinationPath birthday-website.zip -Force
```
